# Loja de Games

API REST para uma loja de games, com produtos, categorias e autenticação de usuários.

## Domínio

- Usuário
- Categoria
- Produto
- Login/autenticação

## Tecnologias

- Java 17
- Spring Boot 2.6.2
- Spring Web
- Spring Data JPA
- Spring Security
- MySQL

## Execução

```powershell
.\mvnw.cmd spring-boot:run
```

Banco configurado por padrão:

```text
jdbc:mysql://localhost/loja-games
```
