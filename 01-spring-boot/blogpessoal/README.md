# Blog Pessoal

API REST de blog pessoal desenvolvida com Spring Boot.

## Domínio

- Usuário
- Tema
- Postagem
- Login/autenticação

## Tecnologias

- Java 17
- Spring Boot 2.6.2
- Spring Web
- Spring Data JPA
- Spring Security
- MySQL, H2 e PostgreSQL
- Springdoc/OpenAPI

## Execução

```powershell
.\mvnw.cmd spring-boot:run
```

Swagger:

```text
http://localhost:8080/swagger-ui.html
```
