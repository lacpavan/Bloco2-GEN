# Farmácia

API REST para gerenciamento de produtos e categorias de uma farmácia.

## Domínio

- Categoria
- Produto

## Tecnologias

- Java 17
- Spring Boot 2.6.3
- Spring Web
- Spring Data JPA
- Bean Validation
- MySQL

## Execução

```powershell
.\mvnw.cmd spring-boot:run
```

Banco configurado por padrão:

```text
jdbc:mysql://localhost/farmacia
```
