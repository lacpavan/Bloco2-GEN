# Modelagem e DER

Esta pasta guarda os arquivos de modelagem do projeto Blog Pessoal.

## Arquivos

| Arquivo | Finalidade |
| --- | --- |
| `blog-pessoal-der/DER_MYSQL_BLOGPESSOAL.mwb` | Modelo editável do MySQL Workbench |
| `blog-pessoal-der/DER_MYSQLBLOGPESSOAL.sql` | Script SQL gerado a partir do modelo |
| `blog-pessoal-der/MYSQL-BLOGPESSOAL.png` | Imagem do DER |
| `blog-pessoal-der/MYSQL-BLOGPESSOALDER.pdf` | Versão em PDF do DER |

## Relação com o código

O DER corresponde ao domínio do projeto `01-spring-boot/blogpessoal`, que trabalha com:

- Usuários
- Temas
- Postagens
